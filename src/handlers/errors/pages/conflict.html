Conflict

HTTP 409
Error Code: conflict

What happened?

The request could not be completed due to a conflict with the current state of the resource. This typically occurs when trying to create or update a resource that already exists or conflicts with existing data.

What can you do?

  • Check if the resource you're trying to create already exists
  • Verify that the data you're submitting doesn't conflict with existing records
  • If updating a resource, ensure you're using the correct version or timestamp
  • Review the error details to understand the specific conflict
  • You may need to retrieve the current state of the resource before making changes
